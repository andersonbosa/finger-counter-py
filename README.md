## Getting started

```bash
# download repository
git clone https://github.com/andersonbosa/finger-counter-py.git

# change directory
cd finger-counter-py

# create virtual environment
python -m venv .venv
source .venv/bin/activate

# install dependencies
pip install -r requeriments.txt

# run project
python main.py
```
